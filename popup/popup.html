LLM Chat Recorder

Inactive
Click the extension icon to start recording
How to use:
  1. Click the extension icon on any LLM chat page
  2. Select text in any chat message
  3. Green outline shows detected chat area
  4. New messages will be broadcast automatically
Broadcast Channel: llm_chat_messages

Your other app can listen with:
new BroadcastChannel('llm_chat_messages')
